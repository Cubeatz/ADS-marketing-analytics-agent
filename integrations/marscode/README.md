# MarsCode MCP 配置

此客户端未使用项目内可安全自动写入的固定配置路径。本脚本已生成可复制的 MCP JSON：

`
C:\Users\Administrator\Projects\marketing-analytics-agent\integrations\marscode\mcp.json
`

使用方式：

1. 打开 MarsCode 的 MCP / 工具 / 服务配置页面。
2. 选择手动添加或 JSON 配置。
3. 粘贴 mcp.json 中的 mcpServers 配置。
4. 保存后重启 IDE，并完成所选平台的 OAuth / API Token 配置。

MarsCode 各版本 MCP 入口差异较大；若当前版本提供 MCP/工具配置入口，请粘贴此 JSON。若没有 MCP 入口，请使用 VS Code / Trae / Qoder / Codex。

只读规则仍以项目根目录 AGENTS.md 为准。
